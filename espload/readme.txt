To build for Macintosh:

  make OS=macosx

  output will be in ../espload-macosx-build/bin


To build for Linux:

  make OS=linux

  output will be in ../espload-linux-build/bin


To build for the RaspberryPi:

  make OS=raspberrypi

  output will be in ../espload-linux-build/bin


To build for Windows:

  make CROSS=win32

  output will be in ../espload-msys-build/bin
